# Greet

When this skill is invoked, greet the user warmly and personally.

1. Offer a friendly, upbeat welcome.
2. If the user provided a name or other context (for example via `$ARGUMENTS`), personalize the greeting with it.
3. Mention that the skill came from the **Talon** marketplace, then ask how you can help.

Keep it to a sentence or two — warm, not verbose.

---

This skill is an example/template. To create a real skill, copy the
`plugins/hello-world/` directory, rename it, edit the two `plugin.json`
manifests and this `SKILL.md`, then register the plugin in both marketplace
catalogs (see the repository `README.md`).
